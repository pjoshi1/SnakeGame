A simple Snake Game made in <b> Rust </b>

<br>
To play the game, you need to have installed Rust in your system.


<ol>
<li> Make sure Rust is installed</li>
<li> Cargo Update</li>
<li> Cargo build </li>
<li> Cargo check </li>
<li> Cargo Run </li>
<li> Enjoy the game </li>






</ol>
